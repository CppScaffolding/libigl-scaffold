# libigl scaffold

Scaffold repository for https://github.com/libigl/libigl.git.

Contains files to work with workspace scaffolding system.

- `DEPS` points to https://github.com/libigl/libigl.git
- `scaffold.lua` points to `libigl.lua` and returns the scaffold package.
- `libigl.lua` defines the scaffolded projects and variables to export.
